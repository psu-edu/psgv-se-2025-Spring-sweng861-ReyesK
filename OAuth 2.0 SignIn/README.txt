This is a simple command line interface used to display a message depending on the method of login. It is only meant to test the functionality of OAuth 2.0.

Getting started:

Step 1 Open the main.py using Python or Python IDE

Step 2 Type "1" to login with Google or "2" to login with LinkedIn.

Step 3 Web browser will ask you to login through your chosen method.

Step 4 After logging in, a message will be displayed to verify a successful login.
